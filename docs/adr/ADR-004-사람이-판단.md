# ADR-004: 해결(Resolved) 확정은 사람이 직접 판단한다

## 상태
Accepted

## 배경
AI 기반 서비스에서는 해결 여부를 자동으로 판단하기 쉽다.
그러나 와르르랩의 목적은 자동 해결이 아니라,
판단의 근거를 남기는 것이다.

## 결정
Resolved 상태는 사람이 직접 확정한다.

- AI는 해결 여부를 단정하지 않는다.
- 사용자가 "이 해결로 문제가 해결되었다"고 판단했을 때만 Resolved를 설정한다.

## 이유
- 해결 여부는 맥락과 판단이 필요한 문제다.
- 자동 판단은 잘못된 결론을 남길 수 있다.
- 사람이 직접 확정한 기록만이 신뢰할 수 있는 자산이다.

## 결과
- Message에 isResolved 필드를 둔다.
- Session은 Resolved 메시지를 기준으로 RESOLVED 상태로 변경된다.
